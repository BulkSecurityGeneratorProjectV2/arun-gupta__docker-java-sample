= Docker Java Sample

This repository contain samples that shows integration of Docker and Java.

== Maven

=== Classical

. Build app: `mvn package`
. Run app: `mvn exec:java`

=== With Docker

. Build app: `mvn package -Pdocker`
. Run app: `mvn install -Pdocker`

== Gradle

=== Classical

. Build app: `./gradlew build`
. Run app: `./gradlew run`
